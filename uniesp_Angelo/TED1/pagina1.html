Assuntos visto na sala

Teremos exemplos de como fazer os assuntos ensinados em sala de aula

  1. Negrito
  2. Itálico
  3. Quebra de Linha
  4. Sublinhado
  5. Sobrescrito
  6. Subscrito
  7. Maior
  8. Menor
  9. Lista

1- Negrito
Temos duas formas de colocar em negrito uma delas é colocando dentro da tag "STRONG" "
A outra é colocando dentro da tag "b"

2- Itálico
Também temos duas formas para itálico, usando dentro da tag "em" e usando dentro da tag "i"

3-Quebra de Linha
Para ir para uma linha abaixo usamos a tag"BR"

4- Sublinhado
Para usar o sublinhado temos duas formas, usando a tag "ins" e a tag "u"

5- Sobrescrito
Para inserir um texto Sobrescrito, usamos a tag "sup". Ai podemos fazer isso: x20+3

6- Subscrito
Para inserir um texto Subscrito, usamos a tag "sub". Ai podemos fazer isso: H2o

7- Maior
Podemos usar a tag "BIG" para deixar um texto um pouco maior, como esta frase. Mas a tag "Big" esta obsoleta

8- Menor
Podemos usar a tag "SMALL" para deixar um texto um pouco maior, como esta frase.

9- Lista

Para lista usamos a Tag "OL" e dentro dela a Tag "li"
Ex:

  1. lista 1
  2. lista 2
  3. lista 3
Podemos mudar a sequencia da lista usando o comando (type="1") dentro do OL, dentro das apas podemos mudar para "A" (fica a sequencia com letras maiusculas) "a"(fica a sequencia com letras minusculas) "I" (fica a sequencia em número romano em maísculo) e por fim "i"(fica a sequencia em número romano em minúsculo)
*OBS: quando digitarmos type="" podemos clicar ctrl + espaco e aparece as opções.

Temos também as listas com a tag "UL", que podemos usar da mesma forma da OL, colando o "IL" dentro das UL
Ex:
  • Sequencia 1
  • Sequencia 2
  • Sequencia 3
Podemos mudar o formato dos pontos antes dos tópicos, colocando (type="circle"), (type="square") ou (type="disc") dentro do UL
