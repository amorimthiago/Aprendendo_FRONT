1- Links
para colocar links usamos a seguinte tag "(a href="pagina2.html">Clique aqui)"O que fica depois das "ASPAS" logo depois que fecha o > do href, é onde ficará o link que será apertado

2- Colocar Fotos
Para colocarmos foto, usamos (img src="") dentro de <>, com isso dentro das "ASPAS" colocamos onde esta a imagem, ou até um endereço para pegar uma imagem em outro site.
Exemplo de foto no site:

3- Favicon
Favicon é a pequena imagem que aparece na aba do navegador ao lado do nome da página. Quando entramos no youtube tem uma foto do lado no nome youtube.
Para colocarmos o favicon, criamos uma linha acima "TITLE que fica no HEAD", com o seguinte código.
(link rel="shortcut icon" href="favicon.ico" type="image/x-icon") Dentro de < >
No caso teremos que criar um arquivo .favicon.ico e deixar na mesma pasta que o index.html

4- Marca Texto
Para usarmos o marca texto, usamos a TAG "mark".
EX:

  1. exemplo de texto com marca texto
  2. Outro exemplo com marca texto.
  3. Mais um exemplo
Para mudar a cor do "Mark" escrevos style="background-color:"" e colocamos a cor dentro das aspas. Será escrito esse código dentro da tag mark.

5- Estilos
Iremos aprender ainda a colocar estilos em um arquivo css, mas também podemos colocar ele no HEAD da página dentro da TAG style. Exemplo de estilo mundando o fundo da tela. (style>body { background-color: cadetblue;} ) Temos que abrir e fechar a tag style
